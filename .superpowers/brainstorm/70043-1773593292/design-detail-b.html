实现细节

1. 渐变边框实现

  • 使用 border-image 或 background-clip 技术
  • 边框颜色: 青色 → 粉色 → 黄色 (赛博朋克三色)
  • 持续脉冲动画 (3s 周期)

2. 图标渐变

  • SVG stroke 使用 linearGradient
  • 暗色主题: 亮色渐变 (#22d3ee → #f472b6 → #fbbf24)
  • 亮色主题: 深色渐变 (#0891b2 → #db2777 → #d97706)
  • 机器人眼睛使用渐变色点缀

3. 动画效果

  • 普通状态: 边框脉冲 (opacity 0.6 → 1, 3s)
  • Hover 状态: 放大 1.05x + 加快脉冲 (1.5s) + 微弱发光
  • Loading 状态: 旋转图标替代脉冲

4. 主题适配

  • 暗色主题: 高对比度,边框颜色更鲜艳
  • 亮色主题: 边框颜色加深,增加阴影层次
  • 支持主题切换时的平滑过渡
